# Northwestern Mutual — Website & Marketing Material Compliance Checklist

*Based on: ADA/Accessibility Guidelines (WCAG 2.1) and Brand Guidelines (Updated March 2026)*

---

## 1. Accessibility (ADA / WCAG 2.1) Contrast Requirements

### Text Contrast
- [ ] Standard body text has a contrast ratio of **at least 4.5:1** against its background
- [ ] Large-scale text has a contrast ratio of **at least 3:1**
- [ ] Text over photographic backgrounds meets 4.5:1 (or 3:1 for large text) — checked across *all* areas of the photo it overlaps, not just one spot
- [ ] If contrast over a photo can't be guaranteed, text is placed on a solid color block instead

**Exceptions (no contrast requirement):**
- [ ] N/A confirmed for: decorative/incidental text, text in inactive UI components, text inside a picture with significant other visual content, text inside a logo

### User Interface Components
- [ ] Text within buttons, links, and interactive components meets **4.5:1**
- [ ] Non-text UI elements (e.g., form input borders, toggles) meet **3:1**

### Graphic & Decorative Elements
- [ ] Purely decorative graphics (no essential meaning) — no strict requirement, but visibility still reviewed

### Logos
- [ ] No contrast ratio required for logos (exempt as logotypes)

### Verification
- [ ] Contrast checked using a contrast-ratio tool before publishing
- [ ] Any borderline color combination cross-checked against WCAG 2.1 documentation

---

## 2. Logo Usage

- [ ] Horizontal logo used (preferred) — stacked version only used when horizontal doesn't fit
- [ ] Correct color version used: Navy on White (preferred), White on Navy, or Black on White
- [ ] Minimum size respected: **1.45" wide (print)** / **160px wide (digital)**
- [ ] Registration mark (®) included if logo is 160px or larger; omitted only if below that size
- [ ] Clear space maintained on all sides (½ the height of the column capital icon)
- [ ] No copy or design elements encroaching on clear space
- [ ] Logo placed bottom-left (preferred), sized so column capital ≈ 65% of margin width
- [ ] Logo NOT recolored, rotated, distorted, given effects (drop shadow, etc.), or split apart
- [ ] Column capital icon used alone **only** for approved cases (social media avatar, app icon) — not general decoration
- [ ] Co-branded/DBA lockups: NM logo equal to or more prominent than partner logo; spacing/hairline specs followed; DBA Guidelines consulted

---

## 3. Color Usage

- [ ] **Primary palette** (Navy, Sky Blue, White) used for anything appearing *outside* owned channels (banner ads, OOH, print, third-party placements)
- [ ] **Secondary palette** (Soft Yellow, Gold, Green, Violet, Cream) used only as backgrounds *within* owned digital channels (e.g., NM.com)
- [ ] Text color correct for background:
  - [ ] Navy or 90% Black (digital) / 100% Black (print) on light backgrounds
  - [ ] White text on Navy, Green, or Violet backgrounds

---

## 4. Photography

**Do:**
- [ ] Calm, uncluttered composition with one clear subject
- [ ] Warm, even lighting
- [ ] Wide depth of field prioritized

**Don't:**
- [ ] Avoid harsh lighting or lens flares
- [ ] Avoid warped/exaggerated angles
- [ ] Avoid multiple angles, high-contrast colors, or visual noise from too many small objects

---

## 5. Typography & Accent Marks

- [ ] Accent marks (highlights, underlines, cross-outs, arrows, checkmarks) used only for emphasis or CTAs, in approved styles (yellow/sky-blue marker look)
- [ ] Logo + tagline lockup (if used) follows correct type sizing/spacing and includes ® on tagline
- [ ] All assets (logos, fonts, icons, illustrations) pulled from the official brand asset library — not recreated or approximated
